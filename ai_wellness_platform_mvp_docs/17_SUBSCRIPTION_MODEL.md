# Subscription Model

## MVP Recommendation

Do not build subscriptions first unless needed for launch.

First prove:
- Users complete onboarding.
- Users use WhatsApp coach.
- Users respond to daily plans.
- Users value weekly reviews.

## Suggested Consumer Pricing

| Plan | Price | Features |
|---|---:|---|
| Free Trial | 7 days | Full access |
| Basic | $19/month | Daily plan, WhatsApp coaching, basic tracking |
| Premium | $39/month | More personalization, weekly/monthly reviews, rewards |
| Family | $79/month | Multiple family members |

## Preferred Long-Term Model

B2B2C may be stronger than direct consumer.

Potential buyers:
- Employers
- Gyms
- Nutrition coaches
- Wellness programs
- Senior centers
- Local health/wellness businesses

## B2B Pricing Examples

| Package | Price |
|---|---:|
| Small group, up to 25 users | $199/month |
| Organization, up to 100 users | $499/month |
| Organization, up to 500 users | $1,500/month |
| Enterprise | Custom |

## Billing Provider

Use Stripe when ready.

Tables:
- subscriptions
- invoices, optional later
- usage_records, optional later

## Subscription Features to Delay

Delay until core product loop works:
- Multiple tiers
- Coupons
- Usage-based billing
- Enterprise invoicing
- Partner billing
- Family account management
