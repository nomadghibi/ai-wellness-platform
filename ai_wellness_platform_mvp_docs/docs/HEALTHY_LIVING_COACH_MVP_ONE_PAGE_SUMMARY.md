# Healthy Living Coach MVP — One Page Summary

## What It Is

A WhatsApp-based AI wellness coach that helps users build healthier daily habits through personalized daily plans, reminders, tracking, rewards, and weekly reviews.

## What It Is Not

It is not a medical app, doctor replacement, diagnostic tool, or treatment platform.

## Core User Promise

Eat better. Drink more water. Walk more. Sleep better. Stay accountable through WhatsApp.

## MVP Core Loop

```text
Profile → Goal → Daily Plan → WhatsApp Check-ins → Tracking → Weekly Review
```

## Main Features

- Web landing page
- User registration/login
- Wellness profile
- Goal selection
- User dashboard
- WhatsApp AI chat
- Daily plans
- Meal/water/activity/sleep/habit logs
- Weekly reviews
- Basic rewards
- Admin portal
- AI safety guardrails

## Build Strategy

Build the reusable AI Wellness Platform Core first, then add the Healthy Living Coach as the first plugin.

## MVP Success Metric

Users respond to the AI coach at least 3 times per week and find weekly reviews useful.
