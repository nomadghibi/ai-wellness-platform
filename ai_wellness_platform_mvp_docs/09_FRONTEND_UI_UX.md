# Frontend UI/UX Design

## Design Principle

The web app is not the main experience. WhatsApp is the main experience.

The web portal should focus on:
- Onboarding
- Profile
- Goals
- Dashboard
- Privacy
- Admin management

## Public Website

### Landing Page Sections

1. Hero
   - Headline: "Your daily wellness coach on WhatsApp"
   - Subheadline: "Build better habits for food, water, walking, sleep, and motivation."
   - CTA: "Start 7-day trial"

2. How It Works
   - Create profile
   - Choose goals
   - Get daily WhatsApp coaching
   - Track progress

3. Features
   - Daily plans
   - Hydration reminders
   - Walking coaching
   - Nutrition guidance
   - Weekly reviews
   - Rewards

4. Safety Statement
   - General wellness only
   - Not medical advice
   - Consult a professional for medical concerns

5. Pricing Preview
   - Free trial
   - Basic
   - Premium
   - Family

6. FAQ
   - Is this medical advice?
   - Does it work only on WhatsApp?
   - Can I delete my data?
   - Can I pause reminders?

7. Final CTA

## User Onboarding Screens

### Screen 1: Create Account
Fields:
- Name
- Email
- Password

### Screen 2: Wellness Disclaimer
Content:
- This is wellness guidance only.
- Not medical advice.
- Do not use for emergencies.
- Consult qualified professionals.

Required:
- Checkbox: I understand and agree.

### Screen 3: Health Profile
Fields:
- Age range
- Height
- Weight
- Goal weight optional
- Activity level
- Food restrictions
- Diet preference

### Screen 4: Goal Selection
Cards:
- Drink more water
- Walk more
- Eat healthier
- Lose weight
- Sleep better
- Build better habits

### Screen 5: Reminder Preferences
Fields:
- Morning plan time
- Hydration reminder on/off
- Walking reminder time
- Weekly review day/time

### Screen 6: Connect WhatsApp
Fields:
- Phone number
- Consent checkbox
- Instructions for opt-in

## MVP Design Style

- Clean
- Calm
- Friendly
- High contrast
- Mobile responsive
- Accessible form labels
- Clear disclaimers
- No clinical/medical look unless compliance is planned
